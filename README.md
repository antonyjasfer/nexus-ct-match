\# NEXUS — Agentic Clinical Trial Matching System

\&gt; Matching patients to life-saving clinical trials in 47 seconds, not 14 months.



\*\*Built for AMD Developer Hackathon 2026\*\*

\- AMD Instinct MI300X (192GB HBM3)

\- Llama 3.1 70B via vLLM on ROCm

\- LangGraph Agent Orchestration



\## Architecture

3-Agent Pipeline:

1\. Protocol Parser — extracts structured eligibility from trial text

2\. Patient Profiler — standardizes medical records for matching

3\. Match Scorer — evaluates fit with explainable reasoning
